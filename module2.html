Unit 1

HTML Tags and Attributes

HTML (HyperText Markup Language) is the standard language for creating web pages. It uses tags to define the structure and meaning of content. Tags are written in angle brackets and most come in pairs (opening and closing).

Attributes provide additional information about an element. They are placed inside the opening tag as name="value" pairs.

<!-- A tag with attributes -->
<a href="https://example.com" target="_blank">Visit Example</a>

<!-- An image tag (self-closing) -->
<img src="photo.jpg" alt="A photo" width="400" />

Unit 2

Structure of an HTML Page

Every HTML document follows a standard structure. The main sections are the DOCTYPE declaration, the <html> root, the <head> (metadata), and the <body> (visible content).

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>My Web Page</title>
    <link rel="stylesheet" href="style.css" />
  </head>
  <body>
    <h1>Hello, World!</h1>
    <p>This is my first web page.</p>
  </body>
</html>

What Each Section Does

  • <!DOCTYPE html> — tells the browser this is an HTML5 document
  • <html lang="en"> — the root element; lang sets the page language for accessibility
  • <head> — contains meta info not shown on screen: charset, viewport, title, stylesheets
  • <meta charset="UTF-8"> — ensures special characters display correctly
  • <title> — the text shown on the browser tab
  • <body> — everything visible to the user goes here

HTML Tables

A table is created with <table>, with rows using <tr>, header cells using <th>, and data cells using <td>.

<table border="1">
  <thead>
    <tr>
      <th>Name</th>
      <th>Age</th>
      <th>Course</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Daniel</td>
      <td>20</td>
      <td>Information Systems</td>
    </tr>
  </tbody>
</table>
TagRoleNotes
<table>Table containerWraps the entire table
<thead>Table headGroups the header row(s)
<tbody>Table bodyGroups the data rows
<tr>Table rowOne row in the table
<th>Header cellBold & centered by default
<td>Data cellRegular table cell

HTML Forms, Multimedia & Creating Full HTML Documents

HTML Forms

Forms allow users to input and submit data. They use the <form> tag with action (where data goes) and method (GET or POST) attributes.

<form action="submit.php" method="POST">

  <label for="name">Full Name:</label>
  <input type="text" id="name" name="name" />

  <label for="email">Email:</label>
  <input type="email" id="email" name="email" />

  <button type="submit">Submit</button>

</form>

🎵 HTML Multimedia — Audio Player Demo

HTML5 introduces native audio support via the <audio> tag. No plugin required.

Code: <audio controls><source src="song.mp3" type="audio/mpeg" /></audio>

Other Multimedia Tags

  • <video> — embeds a video player with controls, autoplay, and loop options
  • <img> — displays images; use alt for accessibility
  • <iframe> — embeds another webpage (e.g. a YouTube video or Google Map)
  • <canvas> — a drawing surface for JavaScript-based graphics
