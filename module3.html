Cascading Style Sheets (CSS)

Bringing life and style to HTML — colors, typography, spacing, and layout control

Unit 1

Basics of CSS

CSS (Cascading Style Sheets) is a stylesheet language used to control the visual presentation of HTML elements — things like color, font, spacing, size, and layout. It separates content (HTML) from presentation (CSS), making web pages easier to maintain.

CSS Rule Syntax

A CSS rule has a selector (which HTML element to style) and a declaration block containing property-value pairs.

/* selector { property: value; } */

p {
  color: #333333;
  font-size: 16px;
  line-height: 1.6;
}

h1 {
  color: navy;
  font-family: Arial, sans-serif;
}

Types of CSS Selectors

SelectorSyntaxWhat It TargetsExample
Element p All <p> elements p { color: red; }
Class .classname All elements with that class .card { padding: 1rem; }
ID #idname One unique element by its ID #header { background: navy; }
Universal * Every element on the page * { box-sizing: border-box; }
Descendant div p <p> inside a <div> nav a { color: white; }
The "Cascading" in CSS: When multiple rules target the same element, the browser follows a cascade order — specificity, source order, and the !important keyword determine which rule wins.
Unit 2

Adding Style to HTML

There are three ways to add CSS to an HTML document:

  1. Inline CSS — applied directly to a single element using the style attribute. Highest specificity but hard to maintain.
  2. Internal (Embedded) CSS — written inside a <style> block in the <head>. Good for single-page styling.
  3. External CSS — stored in a separate .css file and linked via <link>. Best practice for multi-page sites.
<!-- 1. Inline CSS -->
<p style="color: blue; font-size: 18px;">Styled inline</p>

<!-- 2. Internal CSS (inside head) -->
<style>
  body { background-color: #f0f0f0; }
</style>

<!-- 3. External CSS (inside head) -->
<link rel="stylesheet" href="style.css" />

Unit 4

CSS Borders, Margin, Padding & Using CSS in HTML

These three properties control the spacing and boundary of elements. Together they form the CSS Box Model.

The Box Model: Every HTML element is a rectangular box made up of — from inside out — Content → Padding → Border → Margin.
Margin
Border
Padding

  border-radius: 8px;
  padding:       1rem 2rem;
  margin:        1rem auto;
  width:         100%;

}

Margin & Padding Shorthand

  • margin: 10px — all four sides equal
  • margin: 10px 20px — top/bottom | left/right
  • margin: 10px 20px 30px — top | left/right | bottom
  • margin: 10px 20px 30px 40px — top | right | bottom | left (clockwise)
